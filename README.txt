== Description ==

Classic Menus for Block Themes restores the legacy menu management system, allowing you to create and manage multiple menus alongside block-based WordPress themes. This plugin re-establishes the "Appearance > Menus" page and integrates classic menus with the Site Editor.

Features:
- Manage multiple classic menus alongside block-based navigation.
- Easily switch between block-based and classic menu systems.
- Use custom menu IDs or names for flexibility in displaying menus across the site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/classic-menus-for-block-themes` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to Appearance > Menus to create and manage your menus.

== Frequently Asked Questions ==

= Will this plugin disable block-based menus? =
No, this plugin works alongside block-based menus, so you can use both systems.

= Can I add multiple menus? =
Yes, you can create multiple menus through the "Appearance > Menus" screen and use them anywhere in your block theme.
